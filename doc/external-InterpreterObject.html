(abstract) interpreterMethod(text, printDebuggingMessagesopt)

An interpreter method interprets text. When it is called from the outside code, it interprets the text of its first argument. When it is called as a part of another interpreter method, it parses as much of the text as it can, starting from the point in the text where the last interpreter method finished parsing. It returns a type dependent container with the result of itself or of the interpreted methods it is constructed from. It might also be constructed with an interpretation (its parts), in which case it will call that interpretation with the result of its parts and return the result.
Parameters:
Name Type Attributes Description
text string The text to be interpreted.
printDebuggingMessages boolean <optional>
 If this is true, debugging messages are printed to console.log.
