atomInterpretation(parsedText)

An atom interpretation is a callback function passed to InterpreterMethodFactory#atom. Inside the callback, this will refere to the external:ThisBinding. It is called with the string parsed by its external:InterpreterObject#atomTypeInterpreterMethod, which will also return the result of this interpretation.
Parameters:
Name Type Description
parsedText string The text parsed by the interpreter method.
