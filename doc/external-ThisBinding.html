ThisBinding

The interpretation callback functions passed to the methods of an InterpreterMethodFactory when constructing external:InterpreterObject#interpreterMethods for a user created external:InterpreterObject should be thought of as if they were methods of that interpreter object. In practice, that means that this will be bound to the interpreter object inside the body of those interpretations. This class is not a real class whith real methods, but rather a place to collect the documentations of all callback functions that will run as methods of the interpreter object. "So, if they will be run as methods of the external:InterpreterObject, why aren't they documented as methods of that object?" you might ask. This is because an external:InterpreterObject#methodFactoryTypeInterpreterMethod may be used to set the this binding to another object. Therefore, although the callback functions described here will normaly be run as methods of the interpreter object, they are described as if they would be run as methods of some other object refered to as this binding.

atomInterpretation(parsedText) → {*}

An atom interpretation is a callback function passed to InterpreterMethodFactory#atom. Inside the callback, this will refere to the external:ThisBinding. The interpretation is called with the string parsed by its external:InterpreterObject#atomTypeInterpreterMethod, which will also return the result of this interpretation.
