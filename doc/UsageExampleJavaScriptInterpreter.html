Class: UsageExampleJavaScriptInterpreter

UsageExampleJavaScriptInterpreter()

This class is a real world example of how an InterpreterMethodFactory can be used to make a fully functional interpreter. It is an interpreter for a subset of the JavaScript language, just large enough to interpret the source code of InterpreterMethodFactory.js.

It is a fully functional interpreter. It is possible to run all unit tests of Semantics! on this interpretation, and all the tests for this UsageExampleJavaScriptInterpreter can be run on this class being constructed with that interpretation.

Constructor

new UsageExampleJavaScriptInterpreter()

Returns a newly constructed JavaScript interpreter.
